# QRGen 
## Что
Веб-сервис генерации QR-кодов на основе Flask.
## Почему
Потому что захотелось посмотреть, как с помощью Python можно генерировать QR-коды, и создать проект для портфолио.
## Как запустить
`python app.py` в папке с app.py, заходим на `127.0.0.1:5000`. Конфигурация находится в `settings.py`. Перед запуском необходимо определить переменную окружения SECRET_KEY либо командой в терминале, либо в файле `.env`.
